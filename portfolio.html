Projects that incorporate technologies for front end development

The Psychic Game

The Psychic Game is a letter guessing game utilizing javascript. Users will press a letter key and try to guess what the computer has generated.

Crystal Game

The crystal game is a number game that utilizes jQuery. Users will click the crystal images to collect a random number. The objective is to reach the target number. Beware! If you collect too many crystals you lose.

Sailor Moon Trivia Game

Time for a trivia game! you have five seconds to answer each question. how many can you get right?

Giftastic!

Generate random gifs when a button is clicked. Want to see more gifs? Create a new gif button by typing in a new topic.

U-Hike

U-Hike was created as a group project. This web application is a hiking trail locator that allows the user to find a trail once the search button in selected. The user will see several trail options. When they select a trail, the directions for to this hiking trail will display on the map.

Train Schedule

Choo! Choo! Time to board the train! This project utilizes jQuery and firebase to store store and display a train schedule. Users can input information into a form. The information is saved on the database and used to update the train schedule.

Liri App

LIRI is like iPhone's SIRI. However,LIRI is a Language Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data.

Bamazon

bamazon is an Amazon-like storefront. The app will take in orders from customers and deplete stock from the store's inventory.

Burger

Eat-Da-Burger is a restaurant app that lets users input the names of burgers they'd like to eat. Whenever a user submits a burger's name, your app will display the burger on the left side of the page -- waiting to be devoured. Each burger in the waiting area also has a Devour it! button. When the user clicks it, the burger will move to the right side of the page. Every burger will be stored in a database, whether devoured or not.

Friend Finder

This full-stack site will take in results from your users' surveys, then compare their answers with those from other users. The app will then display the name and picture of the user with the best overall match.

Nerd Box

Nerdbox is a semi-customizable "loot box"-style subscription site. Users can select a box they like based on various categories. The user can then checkout as normal.
