The Psychic Game

The Psychic Game is a letter guessing game utilizing javascript. Users will press a letter key and try to guess what the computer has generated.
Crystal Game

The crystal game is a number game that utilizes jQuery. Users will click the crystal images to collect a random number. The objective is to reach the target number. Beware! If you collect too many crystals you lose.

U-Hike

U-Hike was created as a group project. This web application is a hiking trail locator that allows the user to find a trail once the search button in selected. The user will see several trail options. When they select a trail, the directions for to this hiking trail will display on the map.
Train Schedule

Choo! Choo! Time to board the train! This project utilizes jQuery and firebase to store store and display a train schedule. Users can input information into a form. The information is saved on the database and used to update the train schedule.
